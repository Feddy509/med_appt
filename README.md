 # 🏥 Réservation de Rendez-vous Médical
### Medical Appointment Booking Website

---

## 📋 Description du Projet

Ce projet est une application web frontend permettant aux patients de réserver des rendez-vous médicaux en ligne. Elle offre une interface simple, intuitive et responsive pour faciliter la gestion des consultations entre patients et professionnels de santé.

---

## 🚀 Fonctionnalités Principales

- **Prise de rendez-vous** : Sélection de la date, de l'heure et du médecin souhaité.
- **Liste des médecins** : Affichage des spécialistes disponibles par spécialité.
- **Formulaire patient** : Saisie des informations personnelles du patient.
- **Confirmation de rendez-vous** : Récapitulatif et confirmation après réservation.
- **Interface responsive** : Compatible avec les appareils mobiles, tablettes et ordinateurs.
